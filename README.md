# 信息管理
## 技术栈
- 前端
  - Bootstrap 4.4.1
  - jQuery 3.5.1
- 后端
  - node.js 12.16.3
- 数据库
  - MongoDB 4.3.6
- 打包
  - gulp 4.0.2

## 实现功能
支持对学生信息的添加、修改、删除操作
